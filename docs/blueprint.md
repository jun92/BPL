# **App Name**: Blood Pressure Log

## Core Features:

- Blood Pressure Entry: Allow users to enter their systolic and diastolic blood pressure values, along with the time of measurement.
- Data Persistence: Store blood pressure readings, along with the timestamp, in a database.
- Real-time Display: Display blood pressure readings in real-time on the dashboard as soon as they're logged.
- Historical Data List: List blood pressure entries, sorted by timestamp.

## Style Guidelines:

- Primary color: Calming light blue (#A0D2EB) to promote a sense of well-being.
- Background color: Very light, desaturated blue (#F0F8FF) to maintain a clean and readable interface.
- Accent color: A gentle teal (#77D4BB) for interactive elements like buttons, providing a clear visual cue without being alarming.
- Body and headline font: 'PT Sans' for a clean, modern, and easily readable typeface across the application.
- Use clear, simple icons to represent different features or data points within the app.
- Maintain a clean, straightforward layout that makes it easy for users to input and view blood pressure readings, ensuring usability.
- Employ subtle transitions and animations when data is entered or updated, providing smooth user feedback without being distracting.
